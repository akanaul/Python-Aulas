## ⚡ Comandos de Avaliação 1-Clique por Issue

Para rodar qualquer avaliação individual pelo terminal da IDE:

```bash
# Executar teste de uma Issue específica (exemplo Issue #00):
python avaliar_exercicio.py --issue 00

# Executar teste de uma Issue de Mini-Projeto (exemplo #projeto1):
python avaliar_exercicio.py --issue projeto1

# Executar a suíte completa de todas as 25 Issues:
python avaliar_exercicio.py --all
```
