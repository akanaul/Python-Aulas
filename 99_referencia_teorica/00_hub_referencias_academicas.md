# 📚 Hub Central de Referências Acadêmicas & Documentações Oficiais

> [!INFO] Guia Oficial de Fundamentação Teórica e Acadêmica
> Este documento reúne as referências primárias, normas técnicas (PEPs, RFCs, W3C Specs), documentações oficiais de bibliotecas e artigos científicos (papers de IA) que fundamentam o ecossistema do **Curso Python + IA para Automação**.

---

## 🐍 1. Python Core, Tipagem e Boas Práticas (PEPs & CPython Reference)

### Standards da Linguagem (PEPs Oficiais)
- 📘 **[PEP 8 — Style Guide for Python Code](https://peps.python.org/pep-0008/):** O guia supremo de estilo e legibilidade de código em Python.
- 📘 **[PEP 484 — Type Hints](https://peps.python.org/pep-0484/):** Especificação oficial de anotação de tipos em funções e variáveis.
- 📘 **[PEP 572 — Assignment Expressions (Walrus Operator)](https://peps.python.org/pep-0572/):** Expressões de atribuição inline `:=`.
- 📘 **[PEP 428 — The pathlib Module](https://peps.python.org/pep-0428/):** Abstração Orientada a Objetos para caminhos de sistema de arquivos.
- 📘 **[PEP 257 — Docstring Conventions](https://peps.python.org/pep-0257/):** Padronização de documentação de módulos, classes e funções.

### Documentação Oficial CPython
- 📖 **[Python 3.12 Official Documentation](https://docs.python.org/3.12/):** Documentação primária de referência da linguagem.
- 📖 **[Python Data Model & Dunder Methods](https://docs.python.org/3/reference/datamodel.html):** Funcionamento interno dos métodos mágicos (`__init__`, `__repr__`, `__enter__`, `__exit__`).
- 📖 **[Python Standard Library Reference](https://docs.python.org/3/library/):** Módulos nativos (`pathlib`, `json`, `unittest`, `dataclasses`, `asyncio`, `urllib`).

## 📊 2. Manipulação de Dados & Arquivos (Data Engineering Basics)

### Bibliotecas de Manipulação de Dados
- 🐼 **[Pandas User Guide & API Reference](https://pandas.pydata.org/docs/user_guide/index.html):** Documentação oficial para processamento vetorial de DataFrames.
- 📊 **[OpenPyXL Documentation](https://openpyxl.readthedocs.io/):** Manipulação programática de planilhas Excel (`.xlsx`) com suporte a fórmulas e estilos.

### Normas & Especificações de Arquivos
- 🌐 **[RFC 3629 — UTF-8 Standard](https://datatracker.ietf.org/doc/html/rfc3629):** Especificação oficial do formato de codificação de caracteres UTF-8.
- 📧 **[RFC 2045 — Multipurpose Internet Mail Extensions (MIME)](https://datatracker.ietf.org/doc/html/rfc2045):** Estrutura oficial de e-mails em formato HTML e anexos.

## 🤖 3. Inteligência Artificial, Prompt Engineering & Vibe Coding Ético

### Papers Científicos de Referência (arXiv / IEEE)
- 🧠 **[Chain-of-Thought Prompting Elicits Reasoning in Large Language Models (Wei et al., 2022)](https://arxiv.org/abs/2201.11903):** O paper fundador do raciocínio passo a passo em LLMs.
- 🔄 **[ReAct: Synergizing Reasoning and Acting in Language Models (Yao et al., 2022)](https://arxiv.org/abs/2210.03629):** Artigo científico que define o ciclo de raciocínio e execução de ferramentas por Agentes de IA.
- 🎯 **[Language Models are Few-Shot Learners (Brown et al., 2020)](https://arxiv.org/abs/2005.14165):** Fundamentação teórica da aprendizagem in-context com poucos exemplos.

### Ética e Melhores Práticas em IA
- 🛡️ **[Google AI Principles](https://ai.google/responsibility/principles/):** Princípios éticos globais para desenvolvimento e uso responsável de Inteligência Artificial.
- 💻 **[Python Software Foundation Code of Conduct](https://www.python.org/psf/conduct/):** Código de conduta ética na comunidade de desenvolvimento.

## 🌐 4. Automação Web & Desktop (Web Scraping & GUI)

### Especificações W3C e Frameworks Web
- 🌐 **[W3C WebDriver Specification](https://www.w3.org/TR/webdriver2/):** Norma oficial da W3C para automação e controle de navegadores.
- 🚀 **[Selenium 4 Documentation](https://www.selenium.dev/documentation/):** Manual oficial do framework Selenium para controle Dual (DOM + Chrome Options).
- 🔍 **[Chrome DevTools Protocol (CDP) API](https://chromedevtools.github.io/devtools-protocol/):** Especificação técnica dos eventos de rede e inspeção DOM no Chrome.

### Automação de Interface (Desktop)
- 🖥️ **[PyAutoGUI Official Documentation](https://pyautogui.readthedocs.io/):** Controle de mouse, teclado e resolução de tela com failsafe de segurança.
